2 #Components Components

2.1 #Components.form Single row form

_modules/_form.styl

Display, on a single row, an input field + fixed-width icon submit button

Example 
<form class="form-single-row pa1" action="/">
    <input type="text" class="pl1 pr1" placeholder="Input field" />
    <button type="submit" data-icon="search"><em>ok</em></button>
</form>

2.2 #Components.Media Media

_modules/_media.styl

A media is either an image or a video, displayed with caption and/or credit mention. Video: videos are made responsive, on the basis of a 16/9 ratio Empty media: if the media (image or video) is not available for any reason, the block display a default N/A image instead

Example

Image

Video

Empty media

<p>Image</p>
<figure class="media">
    <img src="http://img.20mn.fr/gmiJ0COJRYqUQM6w1jeBlw/648x415_20mn-11765.jpg" alt="This is an image">
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>
<p>Video</p>
<figure class="media media-video-container">
    <iframe width="640" height="360" src="https://www.youtube.com/embed/HwCnti9SW7o" frameborder="0" allowfullscreen></iframe>
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>
<p>Empty media</p>
<figure class="media">
    <span class="media-blank"></span>
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>

2.3 #Components.navbar Navbar

_modules/_navbar.styl

Display a navigation bar at the top of the page. Note that we use headroom.js to handle navbar show/hide behavior on scroll.

Example
Retour à l'accueil
  • Icononly
  • Icon + text
<nav class="navbar" style="position:static">
    <a href="#" class="navbar-brand" data-icon="logo"><em>Retour à l'accueil</em></a>
    <ul class="float-right">
        <li><a href="#" data-icon="user"><em>Icononly</em></a></li>
        <li><a href="#" data-icon="user"> Icon + text</a></li>
    </ul>
</nav>

2.6 #Components.Teaser Teaser

_modules/_borders.styl

Displayed in listings, a teaser is a content summary. It contains:

  • a label
  • a title
  • a summary text
